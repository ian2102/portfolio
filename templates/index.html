Technical Projects

In-Game Item Price Prediction for Dark and Darker

  • Developed an automated item price prediction system for the video game "Dark and Darker."
  • Trained machine learning models using Jupyter and scikit-learn by capturing network data with PyShark and decoding it with Protobuf, tracking over 1 million items.
  • Built a web GUI using Flask, HTML, CSS, and JavaScript, enabling users to submit a form and retrieve accurate price predictions.
  • Integrated image recognition into the web GUI with pytesseract, allowing users to upload item images instead of manually entering data.
